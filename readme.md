# simple C math library
A simple library that helps you do some useless stuff

## API

``` c
#include "sdtarlib"

```

## Usage

```c

int main() {
    int result = add(5, 3);
    printf("5 + 3 = %d\n", result);

    result = subtract(5, 3);
    printf("5 - 3 = %d\n", result);

    return 0;
}

```
